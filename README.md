# Valorant Match Dashboard & Player Analytics

A lightweight web application for tracking Valorant esports match data and visualizing player performance through interactive radar charts.

## Core Features
* **Tournament Schedule:** Real-time updates of regional VCT matches.
* **Match Post-Game Stats:** Detailed KDA, ACS, and ADR breakdown for every player.
* **Player Performance Radar:** Dynamic visualization of player impact and skill sets.

## Technical Stack
* Frontend: HTML5, CSS3, JavaScript (ES6+)
* Data Visualization: Chart.js / Canvas
* API Strategy: Integration with Riot Games Esports API (In Progress)

## Goal
To provide fans with a clean, ad-free environment to analyze match data and player performance without distractions.
