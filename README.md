## English Version

A lightweight web application for tracking VALORANT esports match data and visualizing player performance through interactive radar charts.

### Core Features
* **Tournament Tracking:** Comprehensive schedules and match results for global and regional VCT events.
* **Community Interaction:** A dedicated space for fans to discuss matches and share highlight clips.
* **Match Analytics:** Detailed post-match performance breakdown including KDA, ACS, and economy stats.
* **Pre-match Reference:** An interactive player radar chart tool for fans to analyze and compare player stats before the match starts.

### Technical Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Data Visualization:** Canvas / Chart.js
* **API Integration:** Riot Games VALORANT Esports API (Application in progress)

### Goal
To provide fans with a clean, ad-free, and professional environment to analyze match data and engage with the community.

## 中文版

这是一个轻量级的无畏契约（VALORANT）电竞数据看板，旨在通过交互式雷达图和详细数据可视化展示选手表现。

### 核心功能
* **赛事追踪：** 全球及地区 VCT 赛事的完整日程和比赛结果。
* **社区互动：** 专为粉丝打造的讨论空间，支持分享比赛精彩片段。
* **赛后分析：** 详细的 KDA、ACS（平均战斗得分）和经济数据统计。
* **赛前参考：** 交互式选手数据雷达图，供用户在赛前分析和对比选手能力。

### 技术栈
* **前端：** HTML5, CSS3, JavaScript (ES6+)
* **数据可视化：** Canvas / Chart.js
* **API 接入：** 拳头游戏官方 VALORANT 电竞 API（申请中）

### 目标
为玩家提供一个整洁、无广告、专业的环境来分析比赛数据并参与社区讨论。
